# Billionaire BALLER BOXES

A mobile-first Next.js website and starter application structure for a luxury concierge ordering platform.

## Included
- Landing page
- Bundle tier page
- Registration and onboarding page
- Partner and coupon page
- Operations and delivery proof page
- Shared components and reusable data file
- Tailwind CSS styling

## Local setup
1. Install Node.js 20.9 or newer.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000`
